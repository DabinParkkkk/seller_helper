운송장 정리를 더 쉽게 도와드릴게요

📋 이렇게 사용하세요

  1. 쿠팡 판매자센터 → 발주서 엑셀 다운로드
  2. 택배사 선택 후 파일 업로드
  3. 변환하기 클릭 → 한진 양식 파일 + 정렬된 쿠팡 파일 2개 다운로드
  4. 한진 파일 → 한진 사이트에 업로드 후 출력
  5. 정렬된 쿠팡 파일 → 운송장 대량 등록 페이지에서 사용
💡 사용자 설정을 선택하면 직접 컬럼을 지정해서 다른 쇼핑몰 파일도 변환할 수 있어요. (로그인 후 설정 저장 가능)
